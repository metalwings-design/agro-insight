# Security Policy

## Reporting a Vulnerability

If you find a security bug or vulnerability in this project, *do not open a public issue.* Please follow these steps instead:

1. **Email the Maintainer:** Send a direct email to the project maintainer outlining the problem.
2. **Provide Details:** Include a brief description of the bug and the steps needed to reproduce it.
3. **Allow Time to Fix:** Please give us a few days to fix the issue before sharing it publicly. 

We will review your report within 48 hours and release a patch as soon as possible.
